# ▶️ How to Run the Programs

### Clone the repository

```bash
git clone https://github.com/YourUsername/Cpp_Learning.git
```

### Open the project

Open the folder in **Visual Studio Code**.

### Compile the program

Using g++:

```bash
g++ filename.cpp -o output
```

Example:

```bash
g++ hello.cpp -o hello
```

### Run the program

Windows:

```bash
hello
```

Linux/macOS:

```bash
./hello
```
